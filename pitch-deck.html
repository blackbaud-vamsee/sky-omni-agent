The Problem

Blackbaud developers waste hours on two pain points

📖

SKY UX Doc Fatigue

Engineers context-switch across 35+ component docs, NPM packages, and API references just to scaffold a single form — losing 20–30 min per feature.

🔥

Error Archaeology

When Luminate, RE NXT, or SKY API throws cryptic stack traces, developers spend hours searching Slack, Confluence, and StackOverflow for root causes.

💬 "I spent 3 hours debugging a NULL constituent error that turned out to be a missing @Transactional boundary — something a senior knew in 2 minutes."

AI Usage Disclosure

How GitHub Copilot was used

  • 🤖 Scaffolding — Copilot generated initial Spring Boot controller, CORS config, and Angular component skeletons from brief prompts like "Spring Boot controller for SSE streaming with Flux"
  • 🤖 LangChain4j Integration — Prompted: "LangChain4j AiServices builder with ContentRetriever and Flux streaming return type"
  • 🤖 Knowledge Base Authoring — Copilot assisted writing Blackbaud-specific error pattern .md files from real error descriptions
  • 🤖 Frontend Streaming UI — Prompted: "Angular component consuming SSE stream token by token and appending to a code-highlighted div"
  • ✍️ Human authored — All @SystemMessage prompts (agent personalities), RAG configuration parameters, SKY UX knowledge curation, and hackathon-specific business logic
